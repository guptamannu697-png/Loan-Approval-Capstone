# Python + Machine Learning (Logistic Regression)

This folder contains the complete Python notebook for the Loan Approval prediction model.

## Steps covered
- Data loading (train/test CSV)
- Missing value handling
- One-hot encoding & column alignment
- Train-test split (stratified)
- Model training: Logistic Regression + StandardScaler
- Evaluation: Accuracy, Confusion Matrix, Classification Report, ROC-AUC
- Final submission file generation (submission.csv)
